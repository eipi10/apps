# Interleaved Practice Timer

A browser app for setting up short, repeated practice blocks inspired by interleaved practice routines for musicians and other learners as described in the book *Learn Faster, Perform Better* by Molly Gebrian.

## Features

- Fixed timer blocks such as 3 or 5 minutes.
- Random timer blocks inside a range, such as 3 to 5 minutes.
- A custom list of exercises, passages, excerpts, or sections.
- Ordered rotation through the list.
- Random selection that never repeats the previous item when more than one item is available.
- Responsive interface for desktop, tablet, and phone practice sessions.

## Local Development

```bash
npm install
npm run dev
```

## Tests

```bash
npm run test:all
```

The test suite includes Vitest unit tests for the timer and sequencing rules, a production build check, and Playwright tests for the interactive interface.
